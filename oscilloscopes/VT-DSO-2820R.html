Introduction

VT DSO-2820R is one of the second-generation USB DSOs (USB oscilloscopes) designed and developed by Virtins Technology. This generation of USB DSOs features Virtins Technology’s unique hardware-based DSP algorithm which enhances the performance and functionality dramatically without adding extra hardware cost. When used in conjunction with Multi-Instrument® software, the USB DSO converts any desktop, laptop, or tablet PC into a powerful oscilloscope, spectrum analyzer, multimeter, data logger, signal generator and so forth, all of which work simultaneously.
