Specifications

  1. Support waveform conversion among acceleration, velocity and displacement. Support both SI and Imperial unit systems. This function is provided through the Oscilloscope window.
  2. Display the RMS, Peak/PP, Crest Factor values for acceleration, velocity, displacement if acceleration, velocity or displacement sensors are used. This function is provided through the Multimeter window.
  3. The acceleration, velocity or displacement sensors can be configured through the Calibration window of Multi-instrument.
  4. Supports Multilingual User Interface under Windows 2000, XP, Vista, 7, 8, 8.1, 10, 11 and above. Currently supported languages are English, French, German, Italian, Portuguese, Spanish, Russian, Simplified Chinese, Traditional Chinese, Japanese and Korean.
