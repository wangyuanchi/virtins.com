Software Licensing Guidelines

1. Please ensure that you have downloaded and tried out the fully functional trial version and determined that our software suits your requirements before making the purchase. If you have not done so, please download and try it out here. There is no functional difference between the trial version and the registered version, except that the trial version is only authorized for temporary use for 21 days. For softkey activated license, a unique activation code (softkey), which locks to your computer, will be provided to you within 24 hours via email, fax or phone for you to turn the trial version into a registered version for permanent use after you purchase and register the software. Software upgrade for the same license level and bug fixing is FREE for life.

2. For softkey activated licenses, if you want to use our software on more than one computer, you will need additional licenses. Each computer requires a separate activation code. A USB hardkey option is provided for each license. You can use the USB hardkey on any computer. Both softkey and hardkey are remotely upgradable, which means that you can purchase a lower-level license first and upgrade it later conveniently when necessary.

3. After you make the payment, click here to register or upgrade your softkey activated license, click here to upgrade your hardkey or hardware activated license.
